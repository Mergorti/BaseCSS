Comment utiliser Base CSS

Base CSS n'est pas un framework, mais vraiment une base de travail qui redéfinie le CSS des balises HTML les plus courantes. Les couleurs sont loin d'être séduisante, à vous de les modifiers. Aucune class ne contient l'attribue !important. Il faut un <meta name="viewport" … et un <link rel="… qui importe base.css. Pour reste reportez-vous au code HTML suivant :

<!doctype html>
<html lang="fr">
<head>
	<title>The Project</title>
	<meta charset="UTF-8">
	<meta name="viewport" content="width=device-width, initial-scale=1.0">
	<link rel="stylesheet" type="text/css" href="base.css">
</head>
<body>
	<div id="page">
		<header>
			<h1>The Project</h1>
			<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
		</header>
		<nav>
			<ul>
				<li><a href="home.html">Home</a></li>
				<li><a href="project.html">The Project</a></li>
				<li><a href="about.html">About</a></li>
			</ul>
		</nav>
		<main>
			<h3>The Project</h3>
			<p>Mauris aliquam ullamcorper felis, vitae blandit risus pretium id. 
			Praesent condimentum sapien sed semper bibendum. Quisque a erat nec urna fringilla 
			dapibus quis fringilla dolor. Fusce tempor ornare dolor eget venenatis. Sed ut 
			eros quis nibh pretium euismod. Nam luctus vehicula elit ut congue. Sed tempor, 
			purus sed vestibulum sagittis, eros nisi ullamcorper ante, 
			a scelerisque enim felis id libero.</p>
		</main>
		<footer>
			The Project © 2018
		</footer>
	</div>
</body>
</html>

Englobez tous le code de votre body dans une boite <div id="page">. C'est la seule condition pour une utilisation simple de ce CSS. Si vous souhaitez utilisez un Grid, il faudra utiliser des balises <articles> dans la balise <main class="panneau"> en n'oubliant pas la class panneau.

Pour les tableaux il y a que 2 class à connaîtres, <tbody class="bande"> qui mettra 1 ligne sur 2 en évidence avec un fond un peu plus foncé et <td class="numerique"> à appliquer sur les cellules contenant des données de type numérique.

Changer les couleurs

Les valeurs définient dans :root sont des variables CSS. Pour utiliser une variables CSS il faut utiliser la fonction var() et lui fournir en paramètre le nom complet de la variables. Mais pour une utilisation simple vous pouvez juste modifier les valeurs dans :root.

  • --fond-* définie les couleurs de fond (background);
  • --peaulisse-* définie les polices de caractères;
  • --texte-* définie les couleurs des textes;
  • --lien-* définie les couleurs des liens textuels.
:root {
	--blanc: #ffffff;
	--cendre: #9999b3;
	--antracite: #494963;
	--noir: #292943;
	--alert: #ff3300;

	--fond-entete: #00cca3;
	--fond-page1: #e3e3f6;
	--fond-page2: #eeeeee;
	--fond-sombre: #d3d3e6;
	--fond-claire: #efefff;

	--peaulisse-normale: Times New Roman, Georgia, serif;
	--peaulisse-titre: Arial, Helvetica, sans-serif;
	--peaulisse-technique: monospace;

	--texte-titre: #5b2e9e;
	--texte-normale:#292943;
	--texte-effacer: #604785;
	--texte-demarquer: #494953;

	--lien-menu: #5b2e9e;
	--lien-menu-survoler: #666699;
	--lien: #5b2e9e;
	--lien-survoler: #a3a3c6;
}

/* EXEMPLE */
p {
	color: var(--texte-normale);
}
